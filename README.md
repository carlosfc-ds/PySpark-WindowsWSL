# PySpark Setup with WSL, Java 21, and Python

This guide explains how to configure and run PySpark scripts inside Windows Subsystem for Linux (WSL) using Java SDK 21 and Python.

## Overview

- WSL provides a Linux-based development environment  
- Java 21 is required for Spark  
- PySpark enables running Spark jobs in Python

# Project Structure

## Data

The ```data/``` folder contains a dataset in 3 different versions:

- ``data-setvalued.json``: Data extracted from NYU Langone website via web scraping.
- ``data.json``: Data without set-valued columns.
- ``data.parquet``: Data ready to be used by PySpark.

Navigate through these files to see its details.

## Queries - PySpark scripts
The following scripts were used:
- ``hello_spark.py``: Initial script to test environment runs correctly.
- ``show_data.py``: This script reads the data from the parquet file and prints it.
- ``query_one.py``: Tests operations ``filter()`` and ``spark_sum()``.
- ``query_two.py``: Tests operations ``groupBy()`` and ``spark_avg()``.
- ``query_three.py``: Tests operation ``spark_avg()``. This query could be the real implementation of a real big data scenario that requires computation of float values for a large dataset. This PySpark script solves the problem using distributed computing.

## Output - Results
The files in the ``output/`` folder were generated by redirection of input in Linux. See Additional Details section for more information.

- ``hello_text``: Output from ``hello_spark.py``
- ``query_one_output.txt``: Output from ``query_one.py``
- ``query_two_output.txt``: Output from ``query_two.py``
- ``query_three_output.txt``: Output from ``query_three.py``

# Additional Details

## Prerequisites

- WSL (Ubuntu recommended)
- Python 3.9+
- pip

## Install Java 21

```bash
sudo apt update
sudo apt install openjdk-21-jdk -y
java -version
```

## Virtual Environment (Recommended)

Create isolated Python environment:

```bash
python3 -m venv pyspark-env
source pyspark-env/bin/activate
pip install pyspark
deactivate  # when done
```

## Install PySpark

```bash
pip install pyspark
```

## Run with Output Redirection

Save output to file:

```bash
# Redirect stdout to file
python3 example.py > output.txt
```

## Optional: Interactive Mode

```bash
pyspark
```

## Common Issues

- Ensure `JAVA_HOME` is set correctly  
- Check Java compatibility with Spark version  
- Adjust memory if needed  
- Activate venv before running scripts
